Quickly make tables of descriptive statistics (i.e., counts, percentages, confidence intervals) for categorical variables. This package is designed to work in a tidyverse pipeline, and consideration has been given to get results from R to Microsoft Word Â® with minimal pain.

## Installation

You can install the released version of freqtables from [CRAN](https://CRAN.R-project.org) with:

``` r
install.packages("freqtables")
```

This is a basic example which shows you how to solve a common problem:
